# Procedural Generation of Terrain
Using OpenGL and C++

##### Controls:

	ESC to quit.
	WASD to control camera.
	Z (hold) to show vertices.
	X to render terrain in flat shading.
	C to return to default rendering.
	V to freeze water.
	B to unfreeze water.
